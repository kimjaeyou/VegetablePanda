# Vegetable Panda Project
Kosta 286기 3차 Final Project  

## Summary
- 프로젝트명 : 농산물 판다 (Vegetable Panda)  
- 팀원 : 이정우, 이성민, 강윤성, 이인영, 주영준, 김재영  
- 프로젝트 개요 : 농산물 판다는 Spring Boot 기반의 농산물 라이브 스트리밍 경매 서비스로 생산자와 판매자 모두에게 실시간 온라인 경매 서비스를 제공합니다.
